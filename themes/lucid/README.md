# Lucid

一个追求冷静、理智、真诚、简约的 Hugo 主题。

## 特点

- 极简设计
- 响应式布局
- 优雅的排版
- 快速加载
- 易于定制

## 快速开始

1. 安装主题
```bash
git clone https://github.com/your-username/lucid.git themes/lucid
```

2. 在 `config.toml` 中启用主题
```toml
theme = "lucid"
```
