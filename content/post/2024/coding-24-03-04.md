爱与愁的心痛：连续n个不爽的事，每个事都有一个刺痛值，找出连续m个刺痛值的最小值；
#### 思路
- 输入n和m

- 输入num[n]

- 每个数开始向后遍历m-1个（连带自己共m个数求和）

- 找到和的min
#### 总结
不算困难，但是钻牛角尖了：
```c++
    while(p<=n-m)    //类似滑动窗口的计算
    {
        for(int i=p;i<p+m;i++)

            count+=num[i];
        }
        if(count<min) min=count;
        count = 0;
        p++;
    }
```
